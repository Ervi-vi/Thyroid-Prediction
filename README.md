# 🩺 Thyroid Cancer Recurrence Prediction

Aplikasi Machine Learning berbasis Streamlit untuk memprediksi kemungkinan kekambuhan kanker tiroid menggunakan algoritma Random Forest.

## Fitur

- Home
- Dataset
- Visualisasi
- Prediksi
- Evaluasi Model
- About

## Dataset

Thyroid_Diff.csv

## Algoritma

Random Forest Classifier

## Cara Menjalankan

Install library

```bash
pip install -r requirements.txt
```

Jalankan aplikasi

```bash
streamlit run app.py
```
